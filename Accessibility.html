Accessibility

It is important to me that my website is easily accessible to as many people as possible. Therefore, I have worked hard to design this website in such a way that people with any disabilities or impairments, and people on any device or browser, would be able to enjoy it.

Images

I have included alternative text for all images on this website so that anyone who is visually impaired or using text-only browsers will be able to understand fully what is being displayed on the screen.

Navigation

This website uses descriptive images as links, which also show up as alternative text if images aren't able to be seen.

Links

All links open in the same window.
I have checked all of the links on this website an they all are working correctly.

Layout and Design

This website is layed out in a way where the navigation bar is always located in the left side of the screen, even when you scroll up and down. This gives users accessibility to the navigation bar at any time they are on the website.

Text Size

The text size I used on this website can be resized to fit anyone's needs by using the browser's font-size tool.
