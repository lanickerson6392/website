------------ Exercise 1 -------------

Measurement Converter with User Input

Problem Statement

Write a program that converts feet to inches

Plans and pseudocode

Write an onchange event that calls the feetConverter function when the Convert button is pressed
Write an onchange event that calls the inchConverter function when the Convert button is pressed
Create a Convert button
Create an inchConverter function that converts feet to inches: feet * 12
Create a feetConverter function that converts inches to feet: inches / 12
Output inches or feet, depending on the input

Solution

function inchConverter(){ document.converter.feet.value = document.converter.inch.value / 12 }
function feetConverter(){ document.converter.inch.value = document.converter.feet.value * 12 }

------------ Exercise 2 -------------

Convert the numbers 1-5000 into Roman Numerals

Problem Statement

Write a program that converts the numbers 1-5000 into Roman numerals

Plans and pseudocode

Create a Convert button
Declare var numeralCodes
Assign var numeralCodes to an array for Roman Numerals counting by ones, then tens, then hundreds, and last, thousands, up to 5000
Create a convert(num) function that converts a whole decimal number input between 1-5000 into a Roman numeral
Output a Roman Numeral with a decimal value between 1-5000

Solution

input id="text-input" type="text"
button id="convert-button" onClick="var n = parseInt(document.getElementById('text-input').value);document.getElementById('text-output').value = convert(n);">Convert!
input id="text-output" type="text"
var numeralCodes
numeralCodes = [["","I","II","III","IV","V","VI","VII","VIII","IX"], // Ones
["","X","XX","XXX", "XL", "L", "LX", "LXX", "LXXX", "XC"], // Tens
["","C","CC","CCC","CD","D","DC","DCC","DCCC","CM"], // Hundreds
["","M","MM","MMM","MMMM","MMMMM"]]; //Thousands
function convert(num) {
var numeral = "";
var digits = num.toString().split('').reverse();
for (var i=0; i < digits.length; i++){
numeral = numeralCodes[i][parseInt(digits[i])] + numeral;
}
return numeral;
}
